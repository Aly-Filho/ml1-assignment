## Project Objective
This repository contains the implementation of a standard supervised machine learning classification algorithm from scratch, without the use of scikit-learn. The algorithm is then modified to address a specific data challenge and benchmarked against standard datasets.

* **Selected Algorithm:** T.B.D
* **Selected Challenge:** T.B.D

## Repository Structure
* `/data`: Benchmark datasets used for empirical evaluation.
* `/src`: Open-source based, from-scratch Python implementation of the classification algorithm and our proposed variant.
* `/notebooks`: Contains the ready-to-execute Jupyter notebook to reproduce all benchmark results.
* `/presentation`: PDF slides for the final presentation.
